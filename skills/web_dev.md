# Web Development Skills

## Project Setup
- Use `npm create vite@latest <name>` for React/Vue/Svelte projects
- Use `npx create-next-app@latest <name>` for Next.js
- Use `npm init -y` for plain Node.js projects

## Development Workflow
- `npm install` - Install dependencies
- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm test` - Run tests

## File Structure Conventions
- Keep components in `src/components/`
- Keep pages/routes in `src/pages/` or `src/app/`
- Keep utilities in `src/lib/` or `src/utils/`
- Keep styles in `src/styles/`

## Guidelines
- Write clean, modular components
- Use TypeScript when possible
- Include error handling for API calls
- Write README.md with setup instructions
- Use environment variables for configuration (never hardcode secrets)
