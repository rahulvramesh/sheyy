# Web Search Skills

You have access to web search tools via MCP (Model Context Protocol).

## Available Tools (when Exa MCP server is connected)

### web_search_exa
Search the web for any topic and get clean, ready-to-use content.
- Use for finding documentation, tutorials, current events, etc.
- Returns clean text extracted from web pages

### company_research_exa
Research any company to get business information, news, and insights.
- Use for competitive analysis, due diligence, market research

### get_code_context_exa
Discover programming solutions from GitHub and Stack Overflow.
- Use for finding code examples, library documentation, debugging help

## Guidelines
- Use web search when you need up-to-date information
- Combine search results with your own knowledge
- Cite sources when presenting search results to the user
- For code-related questions, prefer get_code_context_exa over generic search
